## 📄 Resume Builder Web Application

This project is a simple and responsive **Resume Builder** built using HTML, CSS, and JavaScript. It helps users create professional resumes quickly by filling out structured input fields and generating a clean, well-formatted layout.

## 🚀 Features
- Easy-to-use interface for entering personal details, education, skills, and experience  
- Clean and modern resume design  
- Real-time preview of the resume  
- Fully responsive (mobile, tablet, desktop)  
- Download or print resume option  
- Lightweight and fast  

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  

## 📂 Project Structure
- index.html – Main structure  
- style.css – Styling  
- script.js – Functionality  

## 💡 How It Works
1. Open the project in a browser  
2. Enter your details in the form  
3. Preview your resume  
4. Download or print it  

## 🎯 Purpose
This project helps users easily create resumes and is also useful for learning basic front-end development and DOM manipulation.
